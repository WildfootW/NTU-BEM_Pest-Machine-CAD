# Cutting Mechanism

## Log

### 3/30

* 將固定刀具的方式改為一端光軸一端動力，動力端為受力較大的方向

* 將傾斜刀片的角度改大，加長刀片讓一部份重疊，避免碰撞
* 將兩個刀片都放在光軸上，保留另一邊垂直刀片也改斜移動的空間（增加切割力道）

### Initial

兩片切刀將塑膠片切開，一邊垂直一邊傾斜
